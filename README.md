# 🐾 AI 데스크탑 펫 (다마고치 스타일)

바탕화면 위에 항상 떠있는 픽셀아트 AI 펫이에요!

---

## 📦 설치

```bash
# 1. 필수 라이브러리 설치
pip3 install pygame

# 또는 설치 스크립트 실행
chmod +x install.sh
./install.sh
```

## 🚀 실행

```bash
python3 pet.py
```

---

## 🎮 조작법

| 버튼 | 기능 |
|------|------|
| ■ 왼쪽 | 전원 ON / OFF |
| ▶ 가운데 | 다음 메시지 보기 |
| ● 오른쪽 | 메시지 멈추기 & 펫이 춤춤 |
| 드래그 | 창을 화면 어디든 이동 |
| ESC | 앱 종료 |

---

## 💬 펫이 말하는 상황

- **켤 때**: 인사 메시지
- **20분 동안 마우스 안 움직이면**: 잔소리 시작 😂
- **45~110초마다**: 랜덤 메시지 자동 등장
- **▶ 버튼**: 원할 때 바로 메시지 호출
- **전원 끄면**: 잠자는 모드 (ZZZ)

---

## ⚙️ 커스터마이징

`pet.py` 파일에서 메시지를 자유롭게 수정하세요:

```python
MSG_RANDOM = [
    "오늘도 화이팅! ★",
    "밥은 먹었어?",
    # ... 여기에 원하는 메시지 추가!
]

MSG_IDLE = [
    "20분째 멍때리는중",
    # ... 오래 자리 비울 때 나오는 메시지
]
```

### 항상 최상단 (Always-on-top)
pyobjc가 설치되어 있으면 자동으로 항상 최상단에 떠있어요:
```bash
pip3 install pyobjc-framework-Cocoa
```
